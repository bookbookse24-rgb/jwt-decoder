# JWT Decoder

A free online JWT decoder tool that runs entirely in the browser.

## Features

- ✅ Decode JWT tokens instantly
- ✅ View header, payload, and signature
- ✅ Check token expiration
- ✅ Syntax highlighting
- 🔒 100% client-side - tokens never leave your device

## Pro Features (Coming Soon)

- Create/Encode JWTs
- Validate signatures
- Bulk decode
- Save favorite tokens
- Export to file

## Pricing

- **Free**: Decode any JWT
- **Pro** ($9/mo): Create JWTs, validate signatures, bulk operations

## Deploy

Deploy to any static hosting:
- Vercel: `vercel --prod`
- Render.com: Connect GitHub repo, set build command to empty
- Netlify: Drag & drop the folder
